# NBA Draft API

API desenvolvida com Flask, SQLAlchemy e SQLite para simular o Draft da NBA.

## Funcionalidades

- Listar prospects
- Buscar prospect por ID
- Cadastrar prospect
- Remover prospect
- Listar franquias
- Buscar franquia por ID
- Simulação da Lottery

## Tecnologias

- Flask
- SQLAlchemy
- SQLite
- Swagger (Flasgger)

## Executando

pip install -r requirements.txt

python create_db.py

python seed.py

python main.py
